# Note Taker
  ![GitHub license](http://img.shields.io/badge/license-MIT-blue.svg)
  ## Description 
  Note Taker is an application that allows users to create and save notes. Notes can be edited and deleted. This application was built using:
  * HTML
  * CSS
  * JavaScript
  * Express.js
